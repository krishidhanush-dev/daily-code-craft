# SVG (Scalable Vector Graphics)

- SVG are a very common image format on the web.
- They are an incredibly powerful tool for creating high-quality, dynamic images for your websites.

## Lesson Overview:
- What SVGs, vector Graphics, and XML are.
- How to create simple SVGs and ADD then to your website.
- When to use SVGs, and when a different image format would be more appropriate.

- SVGs are a scalable image format.
- Which means they will easily scale to any size and retain their quality without increasing their filesize.
- They are also very useful if you used to create or modify your image programs, because your can change their properties through CSS and JS.

## SVGs are often used for:
- Icons
- Graphs/Charts
- Large, simple images
- Patterned backgrounds
- Applying effects to other elements via SVG filters.
