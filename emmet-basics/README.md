# Emmet Syntax Basics

Learning the core syntax of Emmet to speed up HTML/CSS workflow.

## Key Learnings & Syntax:
* **Child (`>`):** To nest elements. (e.g., `div>ul>li`)
* **Sibling (`+`):** To place elements side-by-side. (e.g., `div+p`)
* **Multiplication (`*`):** To repeat elements. (e.g., `ul>li*3`)
* **Class & ID (`.` and `#`):** To add attributes. (e.g., `div.container`, `h1#title`)
